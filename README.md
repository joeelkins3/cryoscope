# ❄ CryoScope

CryoScope is a full-stack geospatial weather and snow data platform built with React, Node.js, and PostgreSQL + PostGIS. It lets you explore historical and real-time weather data through an interactive Leaflet map with heatmap overlays and a 30-day timeline playback slider, a dashboard with temperature time-series and snowfall charts, a storm archive where you can replay historic blizzards on the map and compare them side-by-side, a query builder for filtering by station, date range, and thresholds with CSV/JSON export, and an insights panel that surfaces long-term warming trends, record days, and forecast accuracy metrics. The app runs fully in demo mode with no backend required — just `cd frontend && npm install && npm run dev`. For live data, copy `backend/.env.example` to `.env`, add your OpenWeatherMap API key, and run `docker-compose up --build` to spin up the full stack including the PostGIS database.
